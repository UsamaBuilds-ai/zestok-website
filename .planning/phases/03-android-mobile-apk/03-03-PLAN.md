    Set up Android APK signing so the app can be installed on any Android device. Create the keystore, configure Gradle to use it, and build the release APK.

    **Step 1 — Create the keystore (RESEARCH.md Code Examples):**

    Run from `mobile/android/app/` directory:
    ```bash
    keytool -genkey -v -keystore stock-management.keystore \
      -alias stock-management-key \
      -keyalg RSA -keysize 2048 -validity 10000 \
      -storetype PKCS12 \
      -storepass changeit \
      -keypass changeit \
      -dname "CN=StockManagement, OU=Development, O=StockManagement, L=City, ST=State, C=PK"
    ```

    **Why use -storetype PKCS12:** PKCS12 is the modern keystore format (JKS is deprecated in JDK 17+). The RESEARCH.md Code Examples specify PKCS12 format.

    **IMPORTANT:** The keystore passwords used here (`changeit`) are placeholders. In a real production scenario, use strong passwords and store them securely. For this MVP phase, the passwords are documented for the developer to update.

    **Step 2 — Ensure .gitignore excludes the keystore:**
    Before committing, verify the keystore file is NOT tracked by git. Create the .gitignore entry first (Step 4), then create the keystore.

    **Step 3 — Update mobile/android/app/build.gradle with signing config (RESEARCH.md Code Examples):**

    Add signingConfigs block inside the `android { }` block:
    ```groovy
    android {
        signingConfigs {
            release {
                storeFile file("../stock-management.keystore")
                storePassword System.getenv("STORE_PASSWORD") ?: "changeit"
                keyAlias "stock-management-key"
                keyPassword System.getenv("KEY_PASSWORD") ?: "changeit"
            }
        }
        buildTypes {
            release {
                signingConfig signingConfigs.release
                minifyEnabled false
                proguardFiles getDefaultProguardFile('proguard-android-optimize.txt'), 'proguard-rules.pro'
            }
        }
    }
    ```
    
    **Why environment variables first, fallback to defaults:**
    - `System.getenv("STORE_PASSWORD") ?: "changeit"` — reads from environment variable first, falls back to placeholder

    **PinScreen.tsx — Wrong PIN flow (D-22):**
    Add explicit error state for HTTP 401 from verifyPin:
    ```typescript
    // In the submit handler, after calling verifyPin:
    try {
      const result = await verifyPin(serverIP, pin);
      if (!result.valid) {
        // Server responded 200 but valid: false (D-23: PIN exists but wrong)
        handleWrongPin();
        return;
      }
      // Success
      setPin('');
      setErrorText(null);
      onSuccess();
    } catch (err: any) {
      if (err.type === 'http' && err.statusCode === 401) {
        handleWrongPin();
      } else if (err.type === 'http' && err.statusCode === 403) {
        handleWrongPin(); // 403 from older API — treat same as 401
      } else if (err.type === 'network' || err.type === 'timeout') {
        setErrorText('Cannot connect to server. Check network connection.');
      } else {
        setErrorText('An unexpected error occurred.');
      }
    }
    ```

    The `handleWrongPin` function:
    ```typescript
    const handleWrongPin = () => {
      const newAttempts = attemptCount + 1;
      setAttemptCount(newAttempts);
      setErrorText('Incorrect PIN');
      setIsShaking(true);
      setTimeout(() => setIsShaking(false), 500);
      
      if (newAttempts >= 3) {
        setIsLocked(true);
        setLockoutRemaining(30);
        setErrorText('Too many attempts. Try again in 30s');
      }
    };
    ```

    **PinScreen.tsx — Lockout timer with countdown (D-10):**
    Enhance the lockout useEffect to show a live countdown:
    ```typescript
    useEffect(() => {
      if (!isLocked) return;
      const interval = setInterval(() => {
        setLockoutRemaining(prev => {
          const next = prev - 1;
          if (next <= 0) {
            clearInterval(interval);
            setIsLocked(false);
            setAttemptCount(0);
            setErrorText(null);
            setPin('');
            return 0;
          }
          setErrorText(`Too many attempts. Try again in ${next}s`);
          return next;
        });
      }, 1000);
      return () => clearInterval(interval);
    }, [isLocked]);
    ```
