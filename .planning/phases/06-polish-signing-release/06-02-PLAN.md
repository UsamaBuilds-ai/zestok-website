    Create the file `mobile/android/keystore.properties` with content (use actual paths/passwords):
    ```properties
    storeFile=C\:/Users/memon/.android/stock-mgmt-release.jks
    storePassword=<generated-store-password>
    keyAlias=stock-mgmt
    keyPassword=<generated-key-password>
    ```
    IMPORTANT: Use forward slashes in the Windows path (not backslashes) to avoid Gradle escape issues.
    Use the password you entered during keytool -genkey. This file will be gitignored.

    Step 4 — Update mobile/android/app/build.gradle (D-64, D-67):
    Add BEFORE the `android {` block:
    ```groovy
    def keystorePropertiesFile = rootProject.file('keystore.properties')
    def keystoreProperties = new Properties()
    if (keystorePropertiesFile.exists()) {
        keystoreProperties.load(new FileInputStream(keystorePropertiesFile))
    }
    ```
    Inside the `android {` block, ADD after `namespace` and before `defaultConfig`:
    ```groovy

        release {
            if (keystorePropertiesFile.exists()) {
                storeFile file(keystoreProperties['storeFile'])
                storePassword keystoreProperties['storePassword']
                keyAlias keystoreProperties['keyAlias']
                keyPassword keystoreProperties['keyPassword']
            }
        }
    }
    ```
    In the `buildTypes { release { ... } }` block, ADD:
    ```groovy

    ```
    In `defaultConfig`, UPDATE versionName from "1.0" to "1.0.0" (D-67).
