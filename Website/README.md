# Zestok Website

Marketing website and customer portal for Zestok Inventory Management.

## Pages

- **Home** — Landing page with features, screenshots, and CTA
- **Buy** — Pricing & NayaPay purchase instructions
- **Download** — Direct download links for Windows .exe and Android .apk
- **Admin** — Password-protected payment confirmation panel

## Deployment

This site is deployed to **Vercel** (linked to this subdirectory).
